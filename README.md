# mmi-lambdas
lambdas

- Setup
    * clone repo    
    * cd repo
    * npm install

- Production Setup
    * clone repo
    * sudo chown -R $USER:$(id -gn $USER) repo
    * npm install

- Requirements
    * linux server
    * node v12 or higher
    * npm v6 or higher
    * pm2 system process management (https://pm2.keymetrics.io/docs/usage/quick-start/)
